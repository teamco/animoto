Person Group Manager

README

This sample web app demonstrates group and person management functionalities in skypeweb, including:
  • Group management:
    • creating a new user-defined group in the group list
    • deleting a user-defined group from the group list
    • renaming a user-defined group
    • adding a DG to the group list
    • removing a DG from the group list
  • Person management:
    • adding a person to a group
    • removing a person from a group
    • removing a person from all groups
For example, follow these instructions to run the scenario of creating a new user-defined group and adding a person to the group:
  1. Load this web app in a browser
  2. Log in
  3. Enter a valid name under 'Group name' and click the 'Create Group' button
  4. Check to see the new group is created and added under "Group list"
  5. Enter a string under "Keyword" and click the "Search" button
  6. Choose a person under "Persons found" and click the "Add Person" button
  7. Check to see the person is added under "Person list"
